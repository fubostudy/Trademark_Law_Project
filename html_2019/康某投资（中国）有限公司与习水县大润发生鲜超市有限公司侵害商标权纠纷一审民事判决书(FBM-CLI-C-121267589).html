康某投资（中国）有限公司与习水县大润发生鲜超市有限公司侵害商标权纠纷一审民事判决书

贵州省遵义市中级人民法院
民事判决书
(2020)黔03民初572号

　　原告：康某投资（中国）有限公司。统一社会信用代码：91310000717854767L。
　　法定代表人：黄明端，公司董事长。
　　委托诉讼代理人：魏圣洁，北京盈科（上海）律师事务所律师。
　　委托诉讼代理人：徐西江，北京盈科（上海）律师事务所律师。
　　被告：习水县大润发生鲜超市有限公司。住所。
　　法定代表人：李波，公司总经理。
　　委托诉讼代理人：杨金刚，贵州名城（习水）律师事务所律师。
　　原告康某投资（中国）有限公司（以下简称康某投资公司）与被告习水县大润发生鲜超市有限公司（以下简称习水大润发公司）侵害商标权纠纷及不正当竞争纠纷一案，本院于2020年9月8日立案受理后，于2020年11月11日公开开庭审理了本案。原告康某投资公司委托诉讼代理人魏圣洁、被告习水大润发公司委托诉讼代理人杨金刚到庭参加了诉讼。本案现已审理终结。
　　原告康某投资公司向本院提出诉讼请求：1、判令被告停止侵害原告第5091186号“大润发"注册商标专用权的行为；2、判令被告停止在字号中使用“大润发"商标的不正当竞争行为；3、判令被告赔偿原告经济损失及合理支出人民币15万元；4、判令被告承担本案诉讼费。事实与理由：“大润发"是由台湾润泰集团投资创办的会员制大型连锁综合超市。原告是台湾润泰集团旗下企业，是“大润发"商标的合法权利人，是“大润发"超市中国地区总部。中国大陆地区的“大润发"超市均为原告所投资设立。大润发自1998年7月在上海开设第一家大型超市以来，截至2020年1月底，己在中国大陆地区成功开设414家综合性大型超市，遍布华东、华北、东北、华中、华南五大区域，拥有十万多名员工和十万多名导购，每天为四百多万位顾客提供服务。原告依法享有第5091186号“大润发"注册商标专用权。上述商标核定使用类别为第35类，经原告合法授权，上述商标被用做零售企业字号、标识，使用“大润发"商标字号的零售企业业绩连续多年在国内名列前茅，“大润发"商标己然具备极高的知名度及社会美誉度，对广大消费者日常生活有巨大影响。经过十几年的使用和经营上述商标己经在全国范围内享有极高知名度，“大润发"成为国内零售行业的顶尖品牌，被国家工商总局以及多家法院认定为驰名商标。被告在习水县家“大润发"超市，超市招牌、促销牌、吊牌、价格标签、购物袋、购物小票、微信收款账号等处使用“大润发"标识。综上，原告认为，第5091186号“大润发"是原告拥有的注册商标，原告拥有商标专用权。涉案商标经过原告的长期使用和宣传，己经具有很强的显著性及非常高的知名度。在原告商标具有较高知名度且为相关公众所知悉的情况下，被告的上述行为违反了《商标法》和有关司法解释的规定，侵犯了原告的商标专用权，使消费者误以为被告与原告之间存在某种联系／关系，误导消费者并扰乱正常的社会秩序。被告作为同行业竞争者，在明知原告及涉案商标知名度的情况下，仍然实施上述侵权行为，侵权恶意明显，给原告造成极其恶劣的影响且侵权获利巨大。原告为维护自身合法权益，特提起诉讼。
　　被告习水大润发公司答辩称：一、原告应承担举证证明被告侵害的商标专用权责任，如举证不利，请求法院驳回原告的全部诉讼请求。二、如法院认定被告侵害了原告的商标权，但被告系经过合法登记过，答辩人由于对法律的错误认识理解而注销店铺的行为表示诚挚道歉。答辩人可配合原告以及法庭，为原告以及社会的营商环境作出相应贡献。三、原告应举证证明因侵权产生的损失，且答辩人认为原告要求赔偿的费用过高且不合理。答辩人的行为不具有较大的社会危害性，也没有给原告造成严重的经济损失，答辩人在规模较小的县城，且处于偏远社区经营超市，有显著性的不同也不会造成消费者误认，从实际经营行为不可能为原告形成不正当竞争，也无严重损害原告的市场声誉的行为。
　　本院组织当事人进行了证据交换和质证。根据当事人陈述和经审查确认的证据，本院认定事实如下：
　　2009年12月28日，上海大润发有限公司获准注册了第5091186号“大润发"商标，注册有效期为2009年12月28日至2019年12月27日，核定使用在第35类服务类别上，包括推销（替他人）、组织商业或广告展览、商业橱窗布置、货物展出等。2013年11月27日，原告经受让取得第5091186号“大润发"注册商标专用权，该商标经续展有效期限至2029年12月27日。2015年1月，“大润发"商标被上海市工商行政管理局评定为上海市著名商标。2016年1月18日，国家工商行政管理总局商标评审委员会作出的商评字［2016］第0000004124号《关于第12111501号“大星发DXF"商标无效宣告请求裁定书》认为：“···可以认定康某公司使用在第35类推销（替他人）服务上的第5091186号‘大润发’商标为驰名商标"。
　　2019年12月18日，被告习水大润发公司经工商登记注册成立，经营范围为预包装食品、散装食品、乳制品（含婴幼儿配方乳粉）、日用品、五金产品、电子产品、服务及鞋帽零售等。
　　2020年7月13日，原告的代理人徐进作为申请人，向联合信任时间戳服务中心申请可信时间戳认证，并取得可信时间戳认证证书。经查验，康某投资公司提交的通过联合信任时间戳服务中心的“权利卫士"取证软件拍摄的视频文件及照片通过了该服务中心的时间戳证书验证。上述文件对应的《可信时间戳认证证书》载明，时间戳证书验证的证据文件是由“权利卫士"客户端产生，“权利卫士"使用自带的拍照、录像、录音功能对客观事实进行取证并实时进行可信时间戳电子证据固化保全，证明文件（或电子数据）自申请时间戳时已经存在且内容保持完整，未被篡改。上述视频文件显示取证人员于2020年7月13日来到被告经营的“大润发生鲜超市"经营场所，可见被告经营场所门头招牌为“大润发生鲜超市"，该超市促销牌、价格标签、购物袋、购物小票、微信收款商户名等突出使用“大润发"字样。
　　上述事实，有公证书、商标注册证、核准商标转让证明、可信时间戳认证证书以及当事人陈述等证据在卷印证，经庭审举证质证，符合证据客观性、真实性、关联性，本院予以确认。
　　本院认为，根据双方当事人的诉辩主张，本案争议焦点是：一、被告习水大润发公司的行为是否侵犯了康某投资公司的注册商标专用权，是否构成不正当竞争；二、被告习水大润发公司是否应当承担民事责任以及如何承担。
　　关于争议焦点一。原告康某投资公司是第5091186号“大润发"注册商标合法权利人，其核定使用类别为35类：货物展出、推销（替他人）等。被告习水大润发公司经营的“大润发生鲜超市"所从事的超市经营符合上述服务方式，与“大润发"商标核定使用类别属于相同服务。习水大润发公司在其超市店招、促销牌、价格标签、购物袋、购物小票、微信收款商户名等使用的“大润发"字样与康某投资公司的“大润发"商标外观近似，且其突出使用“大润发"字样，容易导致消费者对服务来源产生误认。根据《中华人民共和国商标法》第五十七条关于“有下列行为之一的，均属侵犯注册商标专用权：……（二）未经商标注册人的许可，在同一种商品上使用与其注册商标近似的商标，或者在类似商品上使用与其注册商标相同或者近似的商标，容易导致混淆的"之规定，习水大润发公司侵犯了原告康某投资公司的注册商标专用权。康某投资公司持有的“大润发"商标已具有较高知名度，消费者可以通过此商标对超市服务来源进行识别。习水大润发公司与康某投资公司经营的“大润发"超市从事相同服务，其在商户字号和经营场所中使用与康某投资公司具有较高知名度的注册商标相同的文字足以导致他人混淆，根据《中华人民共和国商标法》第五十八条关于“将他人注册商标、未注册的驰名商标作为企业名称中的字号使用，误导公众，构成不正当竞争行为的，依照《中华人民共和国反不正当竞争法》处理。"的规定，其行为违反了《中华人民共和国反不正当竞争法》第六条第（一）、（二）项关于“经营者不得实施下列混淆行为，引人误认为是他人商品或者与他人存在特定联系：（一）擅自使用与他人有一定影响的商品名称、包装、装潢等相同或者近似的标识；（二）擅自使用他人有一定影响的企业名称（包括简称、字号等）、社会组织名称（包括简称等）、姓名（包括笔名、艺名、译名等）；"的规定，其行为构成不正当竞争。
　　关于争议焦点二。根据《中华人民共和国侵权责任法》第十五条关于“承担侵权责任的方式主要有：（一）停止侵害；（二）排除妨碍；（三）消除危险；（四）返还财产；（五）恢复原状；（六）赔偿损失；（七）赔礼道歉；（八）消除影响、恢复名誉。以上承担侵权责任的方式，可以单独适用，也可以合并适用。"的规定，被告习水大润发公司应承担停止侵权、赔偿损失民事责任。关于损失赔偿数额，根据《中华人民共和国商标法》第六十三条第一款、第三款“侵犯商标专用权的赔偿数额，按照权利人因被侵权所受到的实际损失确定；实际损失难以确定的，可以按照侵权人因侵权所获得的利益确定；权利人的损失或者侵权人获得的利益难以确定的，参照该商标许可使用费的倍数合理确定。对恶意侵犯商标专用权，情节严重的，可以在按照上述方法确定数额的一倍以上五倍以下确定赔偿数额。赔偿数额应当包括权利人为制止侵权行为所支付的合理开支。权利人因被侵权所受到的实际损失、侵权人因侵权所获得的利益、注册商标许可使用费难以确定的，由人民法院根据侵权行为的情节判决给予五百万元以下的赔偿。"、《最高人民法院关于审理商标民事纠纷案件适用法律若干问题的解释》第十六条第二款“人民法院在确定赔偿数额时，应当考虑侵权行为的性质、期间、后果，商标的声誉，商标使用许可费的数额，商标使用许可的种类、时间、范围及制止侵权行为的合理开支等因素综合确定。"《中华人民共和国反不正当竞争法》第十七条“经营者违反本法规定，给他人造成损害的，应当依法承担民事责任。经营者的合法权益受到不正当竞争行为损害的，可以向人民法院提起诉讼。因不正当竞争行为受到损害的经营者的赔偿数额，按照其因被侵权所受到的实际损失确定；实际损失难以计算的，按照侵权人因侵权所获得的利益确定。经营者恶意实施侵犯商业秘密行为，情节严重的，可以在按照上述方法确定数额的一倍以上五倍以下确定赔偿数额。赔偿数额还应当包括经营者为制止侵权行为所支付的合理开支。经营者违反本法第六条、第九条规定，权利人因被侵权所受到的实际损失、侵权人因侵权所获得的利益难以确定的，由人民法院根据侵权行为的情节判决给予权利人五百万元以下的赔偿。"的规定，鉴于习水大润发公司于2019年12月才注册登记，且经营情况受新冠肺炎疫情的影响，本院综合考虑原告注册商标的知名度及影响力、被告实施侵权行为的主观故意、地区、地区消费水平以及原告维权合理开支等因素确定被告习水大润发公司应赔偿原告康某投资公司经济损失及合理费用30,000元。
　　综上，根据《中华人民共和国商标法》第五十七条第（二）项、第五十八条、第六十三条第一款、第三款、《中华人民共和国反不正当竞争法》第六条第（一）、（二）项、第十七条、《中华人民共和国侵权责任法》第十五条、最高人民法院《关于审理商标民事纠纷案件适用法律若干问题的解释》第十六条第二款之规定，判决如下：
　　一、被告习水县大润发生鲜超市有限公司立即停止实施侵害原告第5091186号“大润发"注册商标专用权的行为。
　　二、被告习水县大润发生鲜超市有限公司立即停止在其企业名称中使用“大润发"字样。
　　三、由被告习水县大润发生鲜超市有限公司于本判决生效之日起十日内赔偿原告康某投资（中国）有限公司经济损失及合理费用共计人民币30,000元。
　　四、驳回原告康某投资（中国）有限公司的其他诉讼请求。
　　如果未按本判决指定的期间履行给付金钱义务，应当依照《中华人民共和国民事诉讼法》第二百五十三条的规定，加倍支付迟延履行期间的债务利息。
　　案件受理费3,300元，由被告习水县大润发生鲜超市有限公司负担。
　　如不服本判决，可在收到本判决书之日起十五日内向本院或者直接向贵州省高级人民法院递交上诉状，同时，在上诉期限届满后七日内向贵州省高级人民法院预交二审案件受理费，上诉于贵州省高级人民法院。逾期，本判决则发生法律效力。
