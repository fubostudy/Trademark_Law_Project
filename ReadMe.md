# 文件说明

* `html_2013`和`html_2019`是已经爬取下来的判决书HTML文本，分别代表2013年和2019年的商标法。
* `Text_analysis.py`：对所爬取的判决书文本做文本分析，抽取其中所需要字段的程序。
* `Selenium_pkulaw.py`：对北大法宝网站进行selenium模拟点击下载HTML文本的程序。
